# serufer-app

## How to use?

1. Download or clone this repo.

2. Install dependencies.

```js
npm install
// or
yarn install
```

3. Run project on iOS / Android.

```js
 yarn start
```

4. Type y or Y to Do you want to install it globally [Y/n]?.
